# 后端服务使用说明

## 快速安装 systemd 服务

```bash
sudo cp 3d-editor-backend.service /etc/systemd/system/ && sudo chmod 644 /etc/systemd/system/3d-editor-backend.service && sudo systemctl daemon-reload && sudo systemctl enable 3d-editor-backend && sudo systemctl restart 3d-editor-backend && sudo systemctl status 3d-editor-backend
```

## 常用命令

```bash
# 启动
sudo systemctl start 3d-editor-backend

# 停止
sudo systemctl stop 3d-editor-backend

# 重启
sudo systemctl restart 3d-editor-backend

# 查看状态
sudo systemctl status 3d-editor-backend

# 查看日志
sudo journalctl -u 3d-editor-backend -f
tail -f logs/app.log
```

## 卸载服务

```bash
sudo systemctl stop 3d-editor-backend && sudo systemctl disable 3d-editor-backend && sudo rm /etc/systemd/system/3d-editor-backend.service && sudo systemctl daemon-reload
```
