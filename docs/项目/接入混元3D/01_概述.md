# 混元3D接入 - 概述

## 目标

集成腾讯混元3D API，为平台提供AI生成3D模型的能力。

## 功能范围

### 核心功能

- 文生3D：根据文本描述生成3D模型
- 图生3D：根据单张或多视角图片生成3D模型
- 任务管理：提交、查询、取消任务
- 结果存储：自动保存生成的模型到模型库

### 配置管理

- 腾讯云API密钥配置
- 模型生成参数配置（版本、面数、材质等）
- 任务并发控制

## 技术方案

### API集成

- 接口域名：`ai3d.tencentcloudapi.com`
- 认证方式：腾讯云API签名v3
- 支持操作：
  - `SubmitHunyuanTo3DProJob` - 提交任务
  - `QueryHunyuanTo3DProJob` - 查询任务

### 数据流程

```
用户输入(文本/图片) → 提交任务 → 轮询查询 → 下载模型 → 保存到模型库 → 返回结果
```

### 存储策略

- 任务记录存储在数据库
- 生成的模型文件保存到模型库（支持本地/NAS）
- 预览图自动关联

## 限制说明

- 默认并发：3个任务
- 任务有效期：24小时
- 支持模型版本：3.0、3.1
- 输出格式：GLB、OBJ（默认）、可选STL/USDZ/FBX
