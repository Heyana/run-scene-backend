# 需求管理平台 - UI设计说明

## 设计风格

参考飞书任务管理，简洁、高效、专注内容。

## 整体布局

```
+--------------------------------------------------+
|  顶部导航: [任务] [文件] [统计] [需求]              |
+----------+---------------------------------------+
| 左侧栏   |  主内容区（看板/列表）                   |
| 所有任务 |                                       |
| 项目列表 |                                       |
+----------+---------------------------------------+
```

## 主要页面

### 1. 任务看板（参考图1）

```
+--------------------------------------------------+
|  [任务] [文件] [统计] [需求]                       |
+----------+---------------------------------------+
| 所有任务 |  编辑器        进图         长期优化    |
|          |  827个任务     99个任务     15个任务    |
| 项目A    |  +--------+  +--------+  +--------+  |
| 项目B    |  |任务卡片|  |任务卡片|  |任务卡片|  |
|          |  |标题    |  |标题    |  |标题    |  |
|          |  |[标签]  |  |[标签]  |  |[标签]  |  |
|          |  +--------+  +--------+  +--------+  |
|          |  +--------+  +--------+  +--------+  |
|          |  |任务卡片|  |任务卡片|  |任务卡片|  |
|          |  +--------+  +--------+  +--------+  |
|          |     [+]         [+]         [+]      |
+----------+---------------------------------------+
```

布局特点：

- 顶部标签页切换
- 左侧项目/分组导航
- 主区域按任务列表分列
- 每列显示任务数量
- 列底部有"+"按钮添加任务
- 任务卡片简洁，显示标题+标签

### 2. 任务详情侧边栏（参考图2）

```
+--------------------------------------------------+
|  任务标题                                    [×]  |
+--------------------------------------------------+
|  状态          [等待开始 ▼]                      |
|  执行者        [○ 未指配]                        |
|  开始时间      2024年1月8日 17点40分              |
|  结束时间      [2024-01-08 18:00:07]            |
|  备注                                            |
|  优先级        [较低 ▼]                          |
|  标签          0                                 |
|  关联内容                                        |
+--------------------------------------------------+
```

字段设计：

- 状态：下拉选择
- 执行者：头像+名称选择器
- 时间：日期时间选择器
- 备注：文本输入
- 优先级：下拉选择
- 标签：标签选择器
- 关联内容：关联任务列表

### 任务卡片

```
+------------------------+

```

- 白色背景，圆角边框
- 标题灰色字体
- 标签圆角，不同颜色

### 任务列表列

```
编辑器              ... 827个任务
+------------------------+
| 任务卡片1              |
+------------------------+
| 任务卡片2              |
+------------------------+
         [+]
```

- 列标题+任务数量
- 可滚动区域
- 底部添加按钮

### 标签样式

- 圆角矩形
- 浅色背景+深色文字
- 常见标签：
  - 等待开始（灰色）
  - 查无（蓝色）
  - 问题反馈（橙色）
  - 等待开发（黄色）


- 圆形头像

```
  └─ 低
```

## 色彩系统

- 主背景：#f7f8fa

### 文字色

- 主文字：#1f2329
- 次要文字：#646a73
- 禁用文字：#c9cdd4

### 标签色

- 灰色：#f2f3f5 / #646a73
- 蓝色：#e8f3ff / #3370ff
- 橙色：#fff3e8 / #ff6a00
- 黄色：#fffbe8 / #f7ba1e

### 边框色

- 默认边框：#e5e6eb
- 悬停边框：#c9cdd4

## 交互设计

### 拖拽

- 卡片可在列之间拖拽
- 拖拽时卡片半透明
- 目标位置显示虚线占位

### 点击

- 点击卡片：右侧弹出详情侧边栏
- 点击"+"：弹出创建任务表单
- 点击标签：筛选该标签任务

### 悬停

- 卡片悬停：显示边框阴影
- 按钮悬停：背景色变深

### 侧边栏

- 从右侧滑入
- 点击遮罩层关闭
- 支持拖拽调整宽度

## 响应式

### 桌面端（>1200px）

- 左侧栏固定200px
- 主区域自适应
- 侧边栏400px

### 平板端（768px-1200px）

- 左侧栏可收起
- 主区域全宽
- 侧边栏覆盖显示

### 移动端（<768px）

- 左侧栏抽屉式
- 单列显示
- 侧边栏全屏

## 动画

- 侧边栏滑入：300ms ease-out
- 卡片拖拽：跟随鼠标
- 列表加载：淡入
- 标签切换：无动画（即时）
