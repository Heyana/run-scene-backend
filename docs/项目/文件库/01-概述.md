# 文件库功能规划

## 功能定位

文件库用于管理公司内部的各类文档和文件，与资产库的区别：

- **资产库**：面向编辑器，存储图片、视频等媒体资源
- **文件库**：面向公司业务，存储 PDF、录屏、文档等办公文件

## 核心功能

### 文件管理

- 上传/下载文件
- 文件预览（PDF、视频等）
- 文件分类和标签
- 文件搜索和筛选
- 文件版本管理

### 权限控制

- 文件访问权限
- 部门/项目隔离
- 操作日志记录

### 统计分析

- 存储空间统计
- 文件类型分布
- 下载次数统计
- 热门文件排行

## 支持的文件类型

### 文档类

- PDF (.pdf)
- Word (.doc, .docx)
- Excel (.xls, .xlsx)
- PowerPoint (.ppt, .pptx)
- 文本 (.txt, .md)

### 视频类

- MP4 (.mp4)
- WebM (.webm)
- AVI (.avi)
- MOV (.mov)

### 压缩包

- ZIP (.zip)
- RAR (.rar)
- 7Z (.7z)

### 其他

- 图片 (.jpg, .png, .gif)
- 音频 (.mp3, .wav)

## 技术架构

参考资产库的实现模式：

- Model 层：数据模型定义
- Controller 层：API 接口
- Service 层：业务逻辑
- 独立配置：YAML + Config 结构
