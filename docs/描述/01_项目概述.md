# Go Web Server 模板

## 简介

基于 **Gin + GORM + SQLite** 的 Go Web 服务器模板，支持 Air 热重载开发和 Wails 桌面应用双模式启动。

## 技术栈

| 类型   | 技术                |
| ------ | ------------------- |
| 框架   | Gin                 |
| ORM    | GORM                |
| 数据库 | SQLite (纯 Go 驱动) |
| 配置   | YAML + ENV          |
| 日志   | logrus              |
| 热重载 | Air                 |
| 桌面   | Wails v2            |

## 核心特性

- **双模式启动**: Air (开发) / Wails (桌面应用)
- **安全中间件**: 速率限制、IP 黑白名单、攻击检测
- **备份系统**: 定时备份、数据库+CDN 备份、云存储支持
- **嵌入式资源**: embed.FS 静态文件服务
- **API 文档**: Swagger 自动生成

## 快速开始

```bash
# 开发模式
air

# 或直接运行
go run main.go
```

## 默认配置

- 端口: `23347`
- 数据库: `./data/app.db`
- 配置文件: `config.yaml`
