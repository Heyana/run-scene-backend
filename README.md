# 企业名片后台管理系统

## 项目概述
这是一个企业级后台管理系统，支持 wails 和 air 启动。系统为企业提供完整的后台管理功能，包括用户认证、企业管理、微信管理、邀请码配置等核心模块。

## 主要功能
- **用户认证模块**: 内置账号登录、密码修改、会话管理
- **企业管理模块**: 企业信息管理、企业列表展示、搜索和筛选
- **微信管理模块**: 微信账号管理、微信ID和昵称管理
- **邀请码配置模块**: 邀请码生成和管理、使用统计
- **系统配置模块**: 系统参数配置、审计日志记录

## 技术栈
- **后端**: Go + Gin + GORM + SQLite
- **前端**: React/Vue.js (待实现)
- **认证**: JWT + bcrypt
- **启动方式**: Wails (桌面应用) + Air (热重载开发)

## 快速开始

### 使用 Wails 启动
```bash
go run main.go
```

### 使用 Air 启动 (开发模式)
```bash
air
```

## 默认配置
- 端口: 23355
- 默认管理员账户: admin / admin123
- 数据库: SQLite (./data/app.db)
